Upper Penninsula Cybersecurity Institute SIEM

×

This Security Operations Center is something I designed from scratch. The plan is to run four completely isolated network zones so I can launch real attacks, watch them get detected by the SIEM, and investigate them the same way a SOC analyst would. The setup will include a pfSense firewall, a UniFi managed switch, Security Onion, Wazuh, Suricata, and a victim network with Active Directory, Metasploitable, and DVWA. I'll be using Kali Linux, MITRE Caldera, and Sliver C2 to generate realistic attack traffic and then hunt it down in the logs. Every simulation I run will get a full written investigation and every config change will get documented. I'm building this as the foundation for my independent study in cybersecurity operations, aligned with the CySA+ CS0-003 exam.
