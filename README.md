# Tristen Dennis — Cybersecurity Portfolio Website

> Live at: [tristendennis3.github.io](https://tristendennis3.github.io)

## About

This is my personal portfolio website built as part of my CS101 coursework at Northern Michigan University. The site documents my home lab Security Operations Center project, showcasing my hardware, software stack, and cybersecurity skills.

---

## Pages

| Page | Description |
|------|-------------|
| Diagram | Network topology diagram of my SOC home lab with an interactive "About My Home Lab" modal |
| Hardware | Flip card gallery of all hardware components in the lab |
| Software | Full software stack table with hover effects |
| Log Challenge | Interactive Wazuh log analysis challenge using a real brute force attack scenario |
| Documentation | Project documentation and design decisions |

---

## Features

- CSS Grid layout for hardware flip card gallery
- Flexbox navigation bar
- Interactive modal popup
- JavaScript form validation with correct/incorrect answer detection
- Flip cards with unique color coding per hardware component
- Terminal-styled log challenge form
- Site-wide external CSS stylesheet
- Responsive design

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Grid, animations)
- JavaScript (DOM manipulation, event handling)
